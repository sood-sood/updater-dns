# xbox-dns.ru updater
Скрипт для автоматической установки и обновления DNS-серверов. Он получает актуальные адреса с сайта xbox-dns.ru и применяет их к активному сетевому подключению.

## Как установить
- Запусти файл "setup_dns.bat"
- Разреши выполнение от имени администратора
- Скрипт установится в планировщике задач и будет находиться в C:\ProgramData\XboxDNS

Скрипт будет работать автоматически при каждом включении компьютера

## Как удалить
1. Открой Планировщик задач
2. Найди и удали задачу XboxDNS_Updater
3. Удали папку C:\ProgramData\XboxDNS
